Windows安装Linux教程

安装 WSL 和 Ubuntu

  1. 打开命令行，执行WSL --install -d ubuntu。

  2. 若报错，则先执行WSL --install，再重启。

但你大概率会遇到这样的问题。

请看这篇文章，这里说的很详细。

配置 Ubuntu 的账号密码

打开 Ubuntu 的命令行。

按提示，输入账号，密码，再次输入密码确认即可。

直接打开 Linux 命令行

完成第3步后，便可在 Windows 桌面，按住 Shift + 鼠标右键，直接打开 Linux 命令行。
